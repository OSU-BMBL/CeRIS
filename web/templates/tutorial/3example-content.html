Example result illustration


We applied IRIS3 on a single-cell RNA-seq dataset consisting of 90 cells from human embryonic organs and 20,214 genes from Yan, et al.. Default parameters (c=1, o=100, f=0.5) were used for QUBIC biclustering; cell types were predicted using SC3 and evaluated by comparing with the ground-truth labels provided by the paper; CTS-Rs were inferenced using DMINDA based on the clustering result of SC3.


This tutorial will guide you through the output page of IRIS3 in details.

Cell type prediction results

2. A heatmap showing the consensus clustering result from SC3. Both predicted labels and user's labels (if provided) are indicated in the figure. The user can download an emf format file by clicking the topright button.

3. A bar plot showing the silhouette score that measures how similar each cell is to its type compared to other clusters.

4. A Sankey plot showing the convergence and divergence of a two cell groups (ground truth versus SC3 prediction). The width of the arrows shown proportionally to the flow quantity.

Note: if ground-truth cell labels are not provided by the user, the Sankey plot will be omitted.

Following the general results, a CTS cell-gene-regulon heatmap is displayed to cover the gene distribution of CTS-Rs in each cell type and corresponding expression level across all cells.

To explore details about Clustergrammer, please read the documentation.

2. Open the heatmap in a new tab.

3 & 4. Download the gene list for CT1S-Rs. The first column is the CTS-R name, followed by its corresponding genes (tab delimited).

5. Click to show the extension bar for figure copy and row/column sorting.


6. Click to show the enrichment analysis result.

7. The gene composition of CTS-Rs. Genes included in a CTS-R are labeled as green. The genes can be sorted by each CTS-R by double-clicking the CTS-R name. Due to the space limit, at most 15 CTS-Rs are shown in the figure based on the rank of CTS-Rs

8. The log-normalized gene expression heatmap for genes covered in CTS-Rs and all cells. Genes can be sorted by double-clicking the gene names (sort cells by value from high to low), cell names (sort genes by value from high to low), or cell types (sort cells by cell type groups)

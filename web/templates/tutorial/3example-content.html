Example result illustration


We applied IRIS3 on a single-cell RNA-seq dataset consisting of 90 human embryonic stem cells and 20,214 genes from Dr. Yan's study. Default parameters (c=1, o=100, f=0.5) were used for QUBIC biclustering; cell types were predicted using SC3 and evaluated by comparing with the ground truth labels provided by the paper; CTS-regulons were inferenced using DMINDA based on the clustering result of SC3.


This tutorial will guide you through the output page of IRIS3 in details.


General results

1. Your Job ID

2. Click to download a zip file containing the result files. See here for details.

3. General results: including the compatible species matched to the database (currently, only human and mouse data are supported). The number of cells and genes are counted based on the input expression matrix, and genes excluded by the optional filtering step are given (if the user chooses not to perform gene filtering step, show "NA" instead). Cell types are predicted by SC3 and biclusters are identified by QUBIC. If user's cell label is not provided, "Number of provided cell types" will be "NA". In all, the total number of CTS-regulons throughout all cell types are predicted by DMINDA.

The table below shows the number of biclusters and CTS-regulons in each cell type.

4. Cell type prediction: See details below.

5. Job settings: See details below.


Cell type prediction results

1. A table showing the four rand indexes for predicted cell type evaluation.

Note: if ground truth cell labels are not provided by the user, the table will be omitted.

2. A heatmap showing the consensus clustering result from SC3. Both predicted labels and user's labels (if provided) are indicated in the figure. The user can download an emf format file by clicking the topright button.

3. A bar plot is showing the silhouette score that measures how similar each cell is to its type compared to other clusters.

4. A Sankey plot is showing the conserveness and divergence of a two cell groups (ground truth versus SC3 prediction). The width of the arrows is shown proportionally to the flow quantity.

Note: if ground truth cell labels are not provided by the user, the Sankey plot will be omitted.


Job settings

A table showing the number of biclusters and CTS-regulons in each cell type. "Predicted cell type" shows up when using SC3 clusters for SCT-regulon inference, otherwise "Cell lable" shows up to indicate provided cell types.


CTS Cell-Gene-Regulon Heatmap

Following the general results, a CTS cell-gene-regulon heatmap is displayed to cover the gene distribution of CTS-regulons in each cell type and corresponding expression level across all cells.

To explore details about how the Clustergrammer, please read the documentation.

1. Choose which cell type to display.

2. Results and evaluations for the predicted cell types from SC3 in a new page.

3. Download the gene list for CT1S-regulons. The first column is the regulon name and followed by its corresponding genes (tab delimited).

4. Click to show the extension bar for figure copy and row/column sorting.




5. Click to show the enrichment analysis result.

6. The gene distribution of CTS-regulons. Genes included in a regulon is labeled as green. The genes can be sorted by each regulon by double-clicking the regulon name. Due to the space limit, at most 15 regulons are shown in the figure based on the rank of regulons

7. The log-normalized gene expression heatmap for genes covered in CTS-regulons and all cells. Genes can be sorted by double-clicking the gene names (sort cells by value from high to low), cell names (sort genes by value from high to low), or cell types (sort cells by cell type groups)

CTS-regulon details

1. Regulon name.

2. A two-column gene list of CT3S-R1. The left column lists the Gene Symbol, and the right column lists the corresponding Gene ID. Each Gene Symbol or Gene ID is linked to its profile on GeneCards for users to discover more gene details.



3. Click to show the Clustergrammer heatmap for CT3S-R1 individually to give users a local view of regulons. The heatmap is almost the same to the CTS cell-gene-regulon heatmap but remove the regulon part and only shows the expression in this specific cell type.
